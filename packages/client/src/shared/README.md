# Shared Layer

프로젝트 전반에서 재사용되는 **도메인 로직이 없는** 공용 코드입니다.

## 구조
- **api/**: Axios 인스턴스, 인터셉터 등 API 클라이언트 설정
- **lib/**: 날짜 변환, 문자열 포맷팅 등 순수 유틸리티 함수
- **config/**: 환경 변수, 상수, 전역 설정
- **types/**: 전역적으로 사용되는 유틸리티 타입 (도메인 타입 제외)
- **ui/**: (선택사항) 외부 패키지에 없는 프로젝트 전용 공통 UI 컴포넌트

## 규칙
- 이 레이어는 상위 레이어(App, Pages, Widgets, Features, Entities)를 절대 import 하면 안 됩니다.
