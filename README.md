# 🌀 올클 (ALLCLL) – 세종대 수강신청 도우미 Frontend

세종대학교 학생들의 수강신청 경험을 혁신하기 위한 **올클(ALLCLL)** 의 프론트엔드 레포지토리입니다.
학생들의 수강신청 고단함을 줄여주기 위해 수강신청 전반에 걸친 다양한 편의 기능을 제공합니다.

## 🔵 주요 기능

### 1. 실시간 여석 확인 및 알림
수강신청 기간 동안 가장 중요한 여석 정보를 빠르고 직관적으로 제공하며, 놓치지 않도록 알림을 지원합니다.
- **실시간 대시보드**: 여석이 많은 인기 교양 과목 TOP 20 등 주요 현황을 한눈에 파악할 수 있습니다.
- **빈자리 알림 (Pin)**: 원하는 과목을 핀(Pin)으로 등록하면, 여석이 발생했을 때(0석 → 1석 이상) 즉시 브라우저 알림을 받을 수 있습니다.
- **테이블 뷰**: 전체 과목의 여석 정보를 테이블 형태로 상세하게 조회할 수 있습니다.
- **전체 여석 조회**: 전학년 수강신청 기간 직전, 모든 과목의 잔여 여석을 미리 확인하여 전략을 세울 수 있습니다.

### 2. 수강신청 연습 (Simulation)
실제 학교 시스템과 유사한 환경에서 수강신청을 미리 연습해볼 수 있습니다.
- **실전 같은 환경**: 매크로 방지 숫자 입력 등 실제 수강신청 프로세스를 그대로 구현했습니다.
- **결과 분석 리포트**: 연습 종료 후 소요 시간, 성공/실패 여부 등을 분석하여 제공합니다.
- **커스텀 시간표 연동**: 미리 짜둔 시간표를 불러와서 바로 연습에 활용할 수 있습니다.

### 3. 시간표 커스텀 (Timetable)
수강신청 전, 나만의 최적화된 시간표를 미리 계획할 수 있습니다.
- **다양한 일정 관리**: 실제 강의 시간표뿐만 아니라 아르바이트, 개인 공부 등 개인 일정을 함께 등록하여 관리할 수 있습니다.
- **중복 허용**: 시간대가 겹치는 과목도 일단 담아두고 비교하며 최적의 조합을 찾을 수 있습니다.

### 4. 관심 과목 경쟁률 확인
수강신청 전, 과목의 인기도를 미리 파악할 수 있습니다.
- **관심 인원 확인**: 특정 과목을 관심 과목으로 등록한 인원 수를 제공하여 경쟁률을 예측하고 수강신청 전략을 세울 수 있습니다.

## 🔵 프로젝트 구조

이 프로젝트는 메인 애플리케이션과 관리자 페이지, 그리고 UI 라이브러리를 포함하는 **Monorepo** 형태로 구성되어 있습니다.

```plaintext
packages/
├── admin/          # 관리자 대시보드 (크롤러 제어, 로그 확인 등)
├── client/         # 학생용 메인 웹 애플리케이션
├── common/         # 패키지 간 공유되는 공통 컴포넌트 및 유틸리티
├── allcll-ui/      # 올클 서비스 전용 디자인 시스템 컴포넌트 라이브러리
├── sejong-ui/      # 세종대학교 스타일을 반영한 UI 라이브러리
├── e2e/            # Playwright 기반의 E2E(End-to-End) 테스트
├── mock-server/    # MSW 기반 Mock API 서버
└── nginx/          # 배포를 위한 Nginx 설정
```

## 🔵 시작하기

### 사전 준비

- Node.js (^18.0.0)
- pnpm (패키지 매니저)

### 설치

1. 레포지토리를 클론합니다:
   ```sh
   git clone https://github.com/allcll/frontend.git
   cd frontend
   ```
2. 의존성을 설치합니다:
   ```sh
   pnpm install
   ```

### 실행

1. **메인 서비스 (Client)** 실행:
   ```sh
   pnpm run client
   ```
2. **관리자 페이지 (Admin)** 실행:
   ```sh
   pnpm run admin
   ```
